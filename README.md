# Project: Disney
Disney API contains endpoints for Characters, Movies and Genres. Also, user registration and login

Full postman documentation: [Disney](https://documenter.getpostman.com/view/20268380/UzBsHQBQ)

# 📁 Collection: Characters 

## End-point: Get All Characters
Use this endpoint to get all characters.
### Method: GET
>```
>{{url}}/api/v1/characters?name=Benedict Cumberbatch
>```
### Query Params

|Param|value|
|---|---|
|name|Benedict Cumberbatch|


### 🔑 Authentication bearer

|Param|value|Type|
|---|---|---|
|token|{{jwt}}|string|

# 📁 Collection: Authentication 


## End-point: Register
Use this endpoint to register as a new user.
### Method: POST
>```
>{{url}}/api/v1/users/auth/register
>```
### Body (**raw**)

```json
{
    "name": "Lucas",
    "email": "lucas@gmail.com",
    "password": "12345678",
    "passwordConfirm": "12345678"
}
```
